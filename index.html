🧠 What is TRB Tutor App?

TRB Tutor is a mobile + web application built especially for students preparing for the Tamil Nadu Teacher Recruitment Board (PG TRB) exams. Designed for tier-3 and rural districts, it connects institutions, tutors, and students under one platform to practice subject-wise MCQ testszes and track performance.

✅ Built with Tamil Nadu TRB syllabus in mind

  • Updated curriculum
  • Expert tutors
  • Rural-friendly design
